# UBDS3 — Introduction to R (2026)

A 5-day hands-on course covering the core R toolkit for data wrangling, visualization, statistics, and reproducible reporting.

Course material: https://carpentries-incubator.github.io/bioc-intro/

---

## Course Overview

| Day | Topic |
|-----|-------|
| 1 | Tidy data, Intro to R and RStudio |
| 2 | Data Wrangling with `tidyverse` |
| 3 | Data Visualization with `ggplot2` |
| 4 | Reproducible Research and SummarizedExperiment  |
| 5 | Mini-Project   |

---

## Day 1 — Tidy data, Intro to R and RStudio

- Data organisation with spreadsheets
- R and RStudio
- Introduction to R

---

## Day 2 — Data Wrangling 

- Starting with data
- Manipulating and analysing data with `dplyr`

---

## Day 3 — Data Visualization

- Manipulating and analysing data with `dplyr` (cont.)
- Data visualization with `ggplot2`

---

## Day 4 — Next steps: Reproducible Research and SummarizedExperiment

- Reproducible Research: using Rmarkdown to prepare reproducible reports
- Introduction to `SummarizedExperiment` / Bioconductor

---

## Day 5 — Mini-Project 

- Mini-project: students will be able to choose from a set of projects and prepare reproducible reports. 

**Mini-project:**

1. Load a raw dataset
2. Clean and wrangle the data
3. Create visualizations
4. Intepret the results and visualisations 
5. Finalise the project as a R Markdown report
