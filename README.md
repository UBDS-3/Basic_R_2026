# UBDS3 — Introduction to R (2026)

A 5-day hands-on course covering the core R toolkit for data wrangling, visualization, statistics, and reproducible reporting.

---

## Course Overview

| Day | Topic |
|-----|-------|
| 1 | The R Environment & Basics |
| 2 | Data Wrangling with `tidyverse` |
| 3 | Data Visualization with `ggplot2` |
| 4 | Basic Stats & Functions |
| 5 | Mini-Project & Reporting with R Markdown |

---

## Day 1 — The R Environment & Basics

- RStudio interface and R Projects (working directories)
- Variables and data types
- R Markdown as a notebook for code and notes (without deep diving in customization)

**Practical:** Create scripts and run basic math operations and variable assignments.

---

## Day 2 — Data Wrangling (`tidyverse`)

- Importing data (`read_csv`, `read_tsv`)
- Introduction to data frames and tibbles
- Core `dplyr`: `filter`, `select`, `mutate`, `arrange`

**Practical:** Data wrangling exercises with `dplyr`.

---

## Day 3 — Data Visualization (`ggplot2`)

- The grammar of graphics: data, aesthetics (`aes`), and geoms (`geom`)
- Common plot types: scatter plots, boxplots, histograms
- Customising plots: labels, themes, etc.

**Practical:** Produce a plot from a provided dataset.

---

## Day 4 — Basic Stats & Functions

- Summary statistics, `t.test`, and correlation
- Control flow: `for` and `while` loops
- Writing reusable functions

**Practical:** Write a function that accepts a data frame, performs a computation, and returns a `ggplot` graph.

---

## Day 5 — Mini-Project & Reporting

- Introduction to `SummarizedExperiment` / Bioconductor
- Mini-project

**Mini-project:**
1. Load a raw dataset
2. Clean and wrangle the data
3. Run basic statistics
4. Create visualizations
5. Knit to an R Markdown report
